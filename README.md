fp-uttak
================

Uttaksregler for foreldrepenger.
Inneholder tre tjenester for foreldrepenger:
* Opprett stønadskontoer
* Fastsett uttaksperioder
* Søknadsfrist

# Komme i gang

Bygges med maven, java 11
